# API FOR USERS MADE WITH NODE.JS AND MONGODB 

## description:
#### A simple example for users api with login, register and profile. This api was created with node, mongo, jwt, bcrypt and express.

## developed with:
Language: 
 - Javascript

Technologies:
1. MongoDB
2. Node.JS:
    - Express
    - Mongoose
    - Bcrypt
    - JsonWebToken
3. Styling code and tools:
    - Nodemon
    - Husky
    - ConventionalCommits
    - ESlint
    - Prettier
